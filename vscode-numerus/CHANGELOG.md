# Changelog

All notable changes to the Numerus++ extension will be documented in this file.

## [0.1.0] - 2024

### Added
- Initial release
- Syntax highlighting for Numerus++ (.npp files)
- Keywords: DECLARA, EST, SCRIBE, AVTEM
- Operators: ADDIUS, SUBTRAHE, MULTIPLICA, DIVIDE
- Built-in functions: ROMANIZA, EXPRIME
- Roman numeral literal highlighting
- Arabic numeral literal highlighting
- String literal support with escape sequences
- Comment support (NOTA:)
- Auto-closing brackets and quotes
- Language configuration for brackets and comments
